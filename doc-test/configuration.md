# Configuration

Learn how to configure your project.

## Environment Variables

Create a `.env` file:

```
API_KEY=your_api_key
DATABASE_URL=your_database_url
```

## Settings

Configure through the dashboard:

1. Go to Project Settings
2. Update General tab
3. Configure Theme
4. Set up Domain

## GitHub Integration

1. Connect your GitHub account
2. Select repository
3. Choose branch
4. Set docs folder path
